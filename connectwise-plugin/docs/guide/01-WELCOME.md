# Welcome

## What this integration does — in one sentence

**Your clients work in ConnectWise. Your team works in osTicket. This plugin
keeps both sides perfectly in sync — so nobody ever has to ask "which system
has the latest update?"**

## The promise

Whatever happens on one side shows up on the other, looking completely
natural:

| When someone... | The other side sees... |
|---|---|
| Creates a ticket in ConnectWise | The same ticket appears in osTicket — same subject, customer, priority, status, even the attached files |
| Replies from osTicket | A normal reply on the ConnectWise ticket — no robot signatures, no "sent by osTicket" |
| Logs 30 minutes of billable time | 0.50 h, billable, with the right work type — on both sides, to the minute |
| Changes the status or priority | The same change, with the same name, on the other side |
| Uploads a file | The actual file, downloadable on both sides |

And just as important — what the client **never** sees: your internal notes
stay internal, your team structure stays private, and no message ever looks
like it came from a machine.
